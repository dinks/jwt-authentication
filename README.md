# Simple Authentication App

* Create a new user
* Login a user
* Endpoint which requires an autheticated user


## Gems

* `jwt` - [https://github.com/jwt/ruby-jwt](https://github.com/jwt/ruby-jwt)
* `shield` - [https://github.com/cyx/shield](https://github.com/cyx/shield)

## Install

* `bundle install`
* `bundle exec rails s`
